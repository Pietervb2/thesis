\chapter{Results}
\label{chapter:Results}
This section presents the results of the constrained Bayesian optimization applied to a full-day simulation compared with the defined benchmark simulation. Three tolerance values, $\alpha = \beta \in \{0.05, 0.15, 0.25\}$ are considered, each applied to simulations using the four heat demand profiles introduced in Section~\ref{sec::heatdemand}. Before presenting the control results, the simulation parameters and the fixing of two of the control input parameters are discussed.


\section{Simulation parameters}
- besides the constants presented in Section ~\ref{sec::physicalConstants}, the following other simulation parameters are applied: 

T_{\text{ambt}} = 20
T_init_water = 20
T_init_pipe = 20
$\Delta \text{pump}$ = 50 kPa

The network used for the simulations is similair to the network used by Eneco for its in-house steady-state model of the heat distribution within a DCS project. The simulated network represents a 23-floor building, with the floors spaced 3 m apart. At every floor, a heat exchanger is connected to the riser at a distance of 2.5 m from the junction, using DN20 piping. The supply and return risers themselves are divided into three sections with decreasing diameter along the height of the building: DN40 for the lowest six floors, DN32 for the following fourteen floors, and DN25 for the top three floors. The parameters of the four pipe types were presented in Tables~\ref{tab:pipe_dn_common} and~\ref{tab:pipe_dn_geometry}.

For the optimization both weights $w_1$ and $w_2$ are set equal to each other with a value of 1/4181 * 1e-3. Therefore not prioritizing the energy used to heat up the system or the energy used by the heat pump to store it. scaling down the weights made the outcome of the cost function more readable. The starting timestep $k_w$ after which the values of the simulation results are used is set equivalent to 6 hours. It was chosen in this way as this is the time it takes before the sensor temperature within the thermostatic bypass valve has stabilized.
